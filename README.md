# Student Grade Calculator (Java)

This is my first Java project.

It calculates total marks, percentage, and assigns grades based on user input.

## Features
- Takes marks for multiple subjects
- Calculates total and percentage
- Assigns grade using conditions

## Concepts Used
- Scanner (user input)
- If-else statements
- Basic Java logic

## How to Run
1. Compile the program
2. Run the program
3. Enter marks when asked
